# VirtualBox Currency Exchange Demonstration

These are the installation instructions for running the _Currency Exchange_ and _The Answer_ applications from within VirtualBox.

These instructions assume you have access to the file `tutorials.ova`. if you do not, please email s.huckle@sussex.ac.uk.

## 1. Import the VirtualBox Appliance

Select the command **Import Appliance** from the VirtualBox _File_ menu. Select the file `tutorials.ova` from the USB drive supplied.

## 2. Installing the Demo's

For the _Currency Exchange_ demo', follow the instructions in `VBoxCurrencyExchangeInstall.md`. For _The Answer_ demonstration, follow the instructions in `VBoxTheAnswerInstall.md`.
